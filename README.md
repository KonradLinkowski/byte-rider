# js13k2019 #

The beginning of a project for [js13kGames 2019][js13k].

## Development ##

[Node][] versions are managed via [NVM][]. The `start` script will launch a
development server on 127.0.0.1:3000.

```bash
nvm install
npm run start
```

The `build` script will package the project.

```bash
nvm install
npm run build
```
